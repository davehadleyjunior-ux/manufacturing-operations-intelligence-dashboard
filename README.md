# Manufacturing Operations Intelligence Dashboard

AI-powered manufacturing analytics dashboard that analyzes operational datasets,
detects anomalies, generates KPI health indicators, and provides predictive insights.

Features
- Automatic KPI detection
- Excel dataset upload
- Executive summary reports
- Predictive failure forecasting
- AI insight generation

Built with:
- Python
- Streamlit
- Pandas
- Plotly
